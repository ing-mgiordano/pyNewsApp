# News App

Creamos nvo py:
npm create vite@latest
√ Project name: ... pyAppNews
√ Package name: ... pyappnews

Done. Now run:

  cd pyAppNews
  npm install
  npm run dev

npm i axios

En consola: npm install @mui/material @emotion/react @emotion/styled
